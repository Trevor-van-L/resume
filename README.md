Feel free to copy and modify this template for your own resume. Shoot me a message if you do something cool with it.

####How to use:

1. [Install TeXLive][http://www.tug.org/texlive/] for XeTeX

2. In a terminal, `xelatex resume`

#####Acknowledgements
